## What It Does

Kaizen Stealth Script is built for compatible Windows games and centers on two main behaviors: automated target tracking and a visibility mode that changes how your character is perceived in-game. It was created as a compact automation helper, keeping the workflow simple while still covering the core functions the script is meant to provide.

The present version focuses on tighter reaction timing and a smoother enable/disable flow. If you want assisted aiming or a way to experiment with visibility states during gameplay, the script keeps both options easy to access without requiring complicated setup.

## Installation and Use

Grab the newest package from the download link above, then unzip it into its own folder on your Windows machine. Start the game first, and once it is running, launch the script executable with administrator rights so it can access the required hooks.

Example activation sequence:
1. Start your game and load into a match or lobby
2. Open the script folder and double-click `kaizen_loader.exe`
3. Use the default hotkeys (F1 for aimbot, F2 for invisibility) to toggle features

## System Compatibility

Kaizen Stealth Script is intended for Windows 10 and Windows 11. It is aimed at a particular game title that works with the loader's hooking approach. Some anti-cheat systems may flag automation scripts - use it at your own discretion. If the game receives a new patch, the script may need an update before it works as expected.
